# Changelog (Beta)

All notable changes to the beta channel are documented here.
Format follows https://keepachangelog.com/en/1.1.0/

## [1.2.6b] - 2026-06-25

### Added
- Redesigned full-bleed player: cover fills the screen with controls over a scrim, chapter context line, and return/confirm jump-history pills
- Library status tabs on the home screen — All / Listening / Not started / Finished, with live counts
- Online cover search (Google Books with OpenLibrary fallback) from a book's options
- Series name + position and reading-status editing in a book's options sheet
- Logical chapters: long files with no embedded chapters are sliced into ~10-minute chapters for a usable table of contents
- Fine-adjust − / + buttons on the speed and volume-boost sliders

### Changed
- Skip buttons now show the actual configured seconds instead of a fixed number
- Tap-and-hold on a book now only selects; the selection bar floats over the top without pushing the grid down, and a 3-dot menu opens book options when one book is selected
- Tapping a home-screen widget opens the active player, not just the app
- The cover-driven theme now carries across the whole app even while idle, using the last-played book's colours
- Folder scanning uses embedded album tags to split loose multi-book folders and to avoid shattering a single tagged book

## [1.2.5b] - 2026-06-22

### Fixed
- Crash on playback start caused by accessing MediaController on the wrong thread in the position ticker

## [1.2.4b] - 2026-06-22

### Fixed
- Seek bar and position time now update continuously during playback — previously only refreshed on pause/resume

## [1.2.3b] - 2026-06-22

### Fixed
- Scan-audiobooks sheet no longer opens on every app launch — only appears on first launch when no folder is set
- Beta builds now check for pre-release updates only; stable builds check for stable releases only (channels no longer cross-notify)

## [1.2.2b] - 2026-06-22

### Added
- Per-book audio settings: EQ and boost are saved per book and restored on playback — settings no longer carry over between books
- Audio presets are now tab-specific: Speed, Boost, and EQ tabs each show and save their own presets
- Long-press a named preset chip to overwrite its stored value with the current slider position (with haptic feedback)
- Home screen shows last-played book as a resume card after app restart, in the same paused-state UI as an active session
- Long hold a book card (1.5 s) opens Book Options: edit title and author, hide from library, or delete permanently
- Hidden books list in Settings → Library — restore ignored books individually
- Changelog shown in Settings → About, per channel
- Update check button moved to Settings → About

### Changed
- Scan folder button removed from nav bar — scanning now happens automatically on first launch and via pull-to-refresh
- First launch automatically prompts for root audio directory
- Header text removed from home screen

### Fixed
- Volume boost no longer bleeds between books (was only saving EQ per-book before)

### Security
- Cover art files are now hidden from the phone gallery via `.nomedia`
